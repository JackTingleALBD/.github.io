IF + AND + OR - Excel
IF + AND + OR combo is useful when you have 2 or more criteria that need to be evaluated & on one of the criteria you can be situational on what that criteria has to equal, the OR element allows you to widen your filter array on that particular criteria, e.g. the example above the OR element is searching for either of these results: E2>400, M2="TRUE"
=IF(AND(F2>60,OR(E2>400, M2 = "TRUE")),"TRUE","FALSE")
Formula Level: Beginner
