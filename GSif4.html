IMPORTRANGE - GSheets
IMPORTRANGE imports a selected amount of cells/columns from one position to another, it is mostly used to import data from one worksheet to another. Importrange does have limitations so, for large datasets >250,000 cells in use I'd recommend looking into workarounds like apps scripts routine (will cover in a later article), but if you aren't looking at managing a dataset of that size then IMPORTRANGE should be fine to use. The first part of the formula is the custom ID in the url of every google sheet.
=IMPORTRANGE("1F_PcCLOD1eqWHnzihZHo6ifimLSl3lH2kXtDE_fudF4","Pokemon Data!A:D")
