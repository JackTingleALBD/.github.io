IF + AND - Excel
IF + AND combo is useful when you have 2 or more criteria that need to be evaluated before the IF Statement can return a 'True' or 'False' outcome. As you can see in the example, we needed to assess 2 things, first criteria is that the 'HP' is greater than 60 & that the 'Total' value is greater than 400. If both of these requirements match then the formula will return a 'True' otherwise it will be 'False'
Later we will asses how to deal with error responses from formulas (IFERROR).
