IF Statement - Excel
If Statements are a staple in majority of the workings that need to be performed in Excel. If Statements will look into a cell & produce true/false outcome from the evaluation you’re presenting. The above scenerio is looking into a Pokemon dataset in partciular the ‘F’ column or the ‘HP’ column within the table, it is looking to see whether the HP is of a value above 60, if so it will return the true outcome “True”, if not it will return “False”. =IF(F2>60,"True","False")
